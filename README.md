# Chatbot RAG

## Installation

1. Cloner le projet :

```
git clone https://github.com/NolanLefebvre/ChatbotRag.git
```

2. Aller dans le dossier :

```
cd ChatbotRag
```

3. Créer le fichier `.env` en te basant sur `exemple.env`

4. Installer les dépendances :

```
npm install
```

## Lancement

Lancer le projet avec :

```
npm run start
```
